The Problem

Keeping kids focused shouldn't mean
spying on them.

✖ Today's Parental Controls

  • Cloud-dependent, privacy-invasive surveillance tools
  • Static blocklists that kids bypass in minutes
  • Per-device agents that get uninstalled
  • Binary block/allow with zero nuance or context

✔ The StudyGuard Approach

  • Network-level DNS filtering — covers every device automatically
  • Orin nano Evaluates kids' arguments for access fairly
  • Reachy Mini robot gives a physical, friendly presence
  • Vision classifies focus state — no screen recording, no keylogging
  • 100% local on Jetson Orin Nano — no cloud, full privacy

User Flow

What Happens When a Site is Blocked

01 📱

Kid Hits Block

DNS query for youtube.com returns 0.0.0.0 via dnsmasq

02 🎙

Kid Speaks Up

"Hey, can I use YouTube? I need a calculus tutorial."

03 🧠

AI Evaluates

Orin Nano weighs the argument against study stats & context

04 🤖

Reachy Responds

Robot nods yes or shakes no with physical head gestures

05 ✅

Access Granted

Domain added to allowlist for 30 min, then auto-revoked

👍

Granted — Time-Limited

Domain unblocked in dnsmasq for N minutes. Reachy nods. Parent sees grant event in dashboard. Auto-expires.

🚫

Denied — With Reason

Orin Nano explains why ("you've been distracted 40% of the session"). Reachy shakes head. Kid can try a better argument.

03 / 05
Architecture

Four Modules, One Device

📷

Vision Engine

vision.py — every 10s
  • Camera captures via /dev/video1
  • Orin Nano classifies: studying, distracted, phone
  • StateManager tracks focus time
  • MJPEG stream to dashboard
🎙

Voice Agent

voice_loop.py — Whisper + Orin Nano
  • PyAudio capture, silence detection
  • Whisper transcription
  • Conversational Orin Nano agent
  • Parses ACTION_GRANT / ACTION_DENY
🛡

DNS Filtering

argue.py — dnsmasq
  • Writes blocklist.conf & allowlist.conf
  • Blocks to 0.0.0.0 and ::
  • Time-limited allowlist entries
  • Restarts dnsmasq on change
📊

Parent Dashboard

FastAPI — :8000
  • Live camera feed & focus stats
  • DNS traffic log viewer
  • Blocklist & allowlist management
  • Manual grant/deny overrides
